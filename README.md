# Homebrew SuperAgents

Homebrew tap for [SuperAgents](https://github.com/bvsbharat/SuperAgents) - a voice-controlled AI assistant for macOS.

## Installation

```bash
brew tap bvsbharat/superagents
brew install --cask superagents
```

## Features

- Voice-controlled AI assistant
- Dynamic Island-style notch UI
- Multiple AI providers (Claude, GPT, Gemini)
- Screen interaction capabilities
- Memory system for personalization

## Requirements

- macOS 14.0+
- MacBook with notch (recommended)

## Updates

SuperAgents uses Sparkle for automatic updates. The app will check for updates automatically.

## Uninstall

```bash
brew uninstall --cask superagents
```
